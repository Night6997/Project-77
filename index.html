How to set up visual studio

  1. First go to the website to download it.This is the link https://code.visualstudio.com/Download
  2. Click on download then wait for it to finish downloading
  3. Open the download file
  4. Then keep on clicking next untill it asks you to install. Install it and click finish
  5. Now open it up and create your own websites


Console screen

The console screen is used for returning a value. You can check the console screen by right clicking a page and clicking inspect or by clicking ctrl+shift+c. You can do addition, subtracting and many other things on the console screen.
